# 前言

你是否有过这样的体验？

买了很多效率工具，建了很多清单，看了很多"如何高效学习"的视频——但到了真正需要执行的时刻，还是提不起劲，还是拖，还是做了一半就放弃了。

问题不在你不够努力，也不在你意志力不够强。问题在于：**你没有系统。**

## 这本书想解决的问题

大多数效率书告诉你"该做什么"。这本书不同——它告诉你**为什么你知道该做却做不到，以及怎么把"该做"变成"自然而然"**。

本书的核心信念是：

- 系统思维优于意志力
- 设计环境优于强迫自己
- 可持续的小行动优于短暂的高强度冲刺

## 适合谁读

- 大学生、研究生、备考人群
- 刚入职的知识工作者
- 自由职业者与远程工作者
- "看了很多效率内容但执行不起来"的人

## 书的结构与使用方式

这本书分为三个递进式的部分，每一部分都是下一部分的基础。

**【第一部分：基础】**

在学任何技能之前，你必须先建立正确的系统观和目标感。这四章会帮你把"我想做到"变成"我知道怎么做"。

**【第二部分：核心技能】**

基于第一部分的基础，这部分提供了四个最关键的执行技能：专注、抗拖延、阅读和笔记。每一项都有可落地的框架和工具。

**【第三部分：高级技能】**

在具备执行能力之后，这部分教你如何加速学习、应对考试输出、管理能量，以及建立一套可持续的数字生活系统。

## 如何最好地使用这本书

1. **按顺序读**：每一部分都是对下一部分的铺垫
2. **不用读完才开始应用**：读完第一部分后，你就可以开始测试第5-6章的工具
3. **标记和反思**：看到一个方法时，问自己："我现在的情况和这个有多像？我可以做什么不同的选择？"
4. **跟着12周路线图走一遍**：真正的改变来自实践，不来自阅读
